# Тестовое задание Svelte 3 + Typescript + Webpack 5

### Установка, запуск тестов

```bash
npm install
npm test
```

### Сборка dev/prod

```bash
npm run build:dev
npm run build:prod
```

### Web-server [localhost:9000](http://localhost:9000)

```bash
npm start
```

### Очистка

```bash
npm run clean
```
